# United Fillings Compliance Platform - Backend

Backend API for United Fillings Compliance Management System.

## Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose)
- **Authentication**: JWT + bcrypt
- **File Storage**: Cloudflare R2
- **Email**: Brevo (Sendinblue)
